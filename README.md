# 📊 End-to-End Data Analytics Project using Excel, MySQL & Power BI

## 📌 Project Overview

This project demonstrates an end-to-end data analytics workflow using the **Global Superstore** dataset. The objective was to transform raw business data into meaningful insights by performing data cleaning, feature engineering, SQL-based business analysis, and building interactive Power BI dashboards.

The project follows a complete analytics lifecycle—from preparing raw data to presenting actionable business recommendations through interactive dashboards.

---

# 🎯 Project Objectives

* Clean and prepare raw transactional data for analysis.
* Perform business-focused analysis using SQL.
* Build interactive and executive-friendly Power BI dashboards.
* Generate meaningful business insights for data-driven decision-making.
* Apply best practices in data visualization and storytelling.

# 📅 Project Journey

## ✅ Day 1 – Data Cleaning & Feature Engineering (Excel)

* Cleaned and validated raw data.
* Standardized date formats.
* Created business metrics including:

  * Processing Time
  * Profit Margin %
  * Revenue per Quantity
  * Profit Status
  * Sales Categories
* Integrated Returns and People tables using VLOOKUP.
* Identified and documented data quality issues.

---

## ✅ Day 2 – SQL Business Analysis (MySQL)

Performed business analysis by writing **30 SQL queries** covering:

* Business KPIs
* Sales Analysis
* Product Performance
* Customer Analysis
* Regional Analysis
* Shipping & Operations
* Returns & Discount Analysis

Validated the imported dataset before analysis and documented business insights for every query.

---

## ✅ Day 3 – Executive Dashboard (Power BI)
